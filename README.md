# 🎥 YouTube Clone

## 📌 Project Overview

This project is a responsive YouTube-inspired web application developed using HTML and CSS. It replicates the basic user interface and user experience of YouTube, including video thumbnails, navigation menus, search functionality, and responsive layouts.

## 🚀 Features

* Responsive Design
* YouTube-style Homepage
* Video Thumbnails
* Navigation Sidebar
* Search Bar Interface
* Mobile-Friendly Layout
* Modern UI Design

## 🛠 Technologies Used

* HTML5
* CSS3
